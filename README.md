# 情報工学科とは
情報工学科とは「__*コンピュータの基礎から、高度なプログラミングまで。情報化社会をさらに進化させる、IT業界の新たなリーダーを育成する。*__」科目でプログラミング教育を中心に3つのコースに分かれ着実な知識を身につけ、卒業研究で主体性や、課題に対する洞察力、コミュニケーション能力、社会で役に立つための総合的能力を習得します。
###主に取れる資格
1. 基本情報技術者試験
2. 応用情報技術者試験
3. プロフェッショナル試験［（独）情報処理推進機構］
4. 色彩検定（１級・２級・３級）［（公社） 色彩検定協会］
5. ITパスポート
## 選べる３つのコース
情報工学科では２年次から*3つのコース*に分かれてより専門的な学びを深めます。
* __システムエンジニアリングコース__
*〜プログラミングによって時代の求める新しい情報システムを作る〜*

プログラミングを基礎から学び、コンピュータを制御するソフトウェアや情報システムの技術を修得します。それにより、時代に必要とされる情報システムを創造するエンジニアを育成します。
* __インターネット&セキュリティーコース__
_〜安全で安心できる持続的な情報社会を支えるネットワークとセキュリティを学ぶ〜_
情報化社会は、私たちの暮らしをとても豊かなものにしてきましたが、様々な脅威も発生しています。それらから私たちの暮らしを守り、安全で安心できる情報社会を作るために、ネットワーク技術や情報セキュリティ対策などについて学びます。
* __知能情報コース__
*〜「モノ」と「コト」の未来へ AIの先を拓く〜*
あらゆるモノがネットワークに繋がるIoT、さまざまなコトに繋げる技術やこれからの未来を支える人工知能は、既に私たち人間にとって日常の一部になっています。これらの知識・技術を基礎から応用まで学び、目覚しく変化する情報社会の新しい今後を創り出す専門家の育成を目指します。
